# فكر مشوش 🌙

### محرك المحتوى الإسلامي الذكي

**Fekr Moshawash** - تطبيق ويب تفاعلي لإنشاء ومشاركة محتوى إسلامي متنوع: آيات قرآنية، أحاديث نبوية، مقولات عميقة، وأدعية مع واجهة تصميم تحاكي تطبيق Notes على iOS.

---

## ✨ الميزات الرئيسية

### 🔄 جلب المحتوى الذكي
- 📖 **آيات قرآنية عشوائية**: تتصل بـ API Alquran.cloud للحصول على آيات حقيقية من القرآن الكريم
- 📜 **أحاديث نبوية وقدسية**: من قاعدة بيانات محلية تتضمن أحاديث مختارة
- 💡 **مقولات عميقة**: خواطر وحكم من أعلام الفكر والحكمة
- 🤲 **أدعية وأذكار**: مجموعة متنوعة من الأدعية المأثورة

### 🎨 تصميم احترافي
- **واجهة تحاكي iPhone Notes**: تصميم عصري وانغماسي
- **الوضع الداكن**: حماية للعيون وتجربة بصرية مريحة
- **دعم النصوص العربية الكامل**: RTL (من اليمين إلى اليسار)
- **خطوط عربية جميلة**: استخدام خطوط Amiri و Cairo للمظهر الاحترافي

### 🛠️ أدوات تحكم متقدمة
- 📝 تحرير النص الرئيسي والخاطرة والمصدر
- 📏 التحكم الديناميكي بأحجام الخطوط
- 🎯 معاينة مباشرة في الوقت الفعلي
- 📸 التقاط صور بجودة HD (3x)
- 💾 تحميل الصور تلقائياً

### 📱 دعم تطبيق الويب التقدمي
- **PWA Support**: تثبيت كتطبيق على الهاتف
- **Offline First**: عمل بدون اتصال بالإنترنت
- **Service Worker**: تخزين مؤقت ذكي
- **Responsive Design**: يعمل على جميع الأجهزة

  - JavaScript ES6+

  - Web Fonts
  - Canvas API

### التثبيت والتشغيل

#### 1. نسخ المستودع
```bash
git clone https://github.com/za401820-ui/fekr-moshawash.git
cd fekr-moshawash
```

#### 2. فتح الملف
```bash
# استخدام أي خادم محلي
python -m http.server 8000
# أو
npx http-server
# ثم افتح في المتصفح: http://localhost:8000
```

#### 3. استخدام GitHub Pages
```bash
# الملف متوفر على:
https://za401820-ui.github.io/fekr-moshawash
```

## 💻 كيفية الاستخدام

### 1️⃣ جلب محتوى عشوائي
اضغط على أحد الأزرار الثلاث:
- **📖 آية عشوائية**: تجلب آية من API الإنترنت
- **📜 حديث نبوي**: تجلب حديثاً من قاعدة البيانات
- **💡 مقولة عميقة**: تجلب خاطرة حكيمة

### 2️⃣ تعديل النصوص
- عدّل **النص الرئيسي** (الآية أو المقولة)
- أضف **الخاطرة/التفسير** (التعليق الشخصي)
- حدّد **المصدر** (من أين جاءت المحتوى)

### 3️⃣ التحكم بالحجم
- اسحب مفتاح **حجم النص الرئيسي**
- اسحب مفتاح **حجم الخاطرة**

### 4️⃣ حفظ الصورة
اضغط على **📸 التقاط الصورة (HD)** لتحميل الصورة
